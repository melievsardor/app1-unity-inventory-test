# App1 Unity Inventory Test

Test assignment implementation for App1 game studio.

## Unity Version
2022.3.62

## Platform
Mobile (Portrait)

## Features

Inventory system with 30 slots  
First 15 unlocked, next 15 purchasable with coins

Scrollable grid using Optimized ScrollView Adapter

Item types:
- Ammo
- Weapons
- Head equipment
- Torso equipment

Ammo stacking supported

Drag and drop supported

Item swapping supported

Popup item info supported

Inventory weight calculation

Coins system

Save system implemented using Easy Save 3

All static data configured using ScriptableObjects.

Architecture follows SOLID principles.

## Controls

Buttons:

Shoot  
Add Ammo  
Add Item  
Remove Item  
Add Coins

## Save System

All data persists between sessions using Easy Save.

Saved data:

Coins  
Inventory contents  
Unlocked slots  
Stacks  

## Bonus Features

Drag and Drop  
Ammo stacking via drag
